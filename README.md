# React Components

Production-grade UI components built with React and TypeScript.

Each component is a rewrite of a Vanilla JS component originally developed
for enterprise clients (Banco BPI, Ericsson, Fidelidade) — translated to
idiomatic React with hooks, TypeScript, and clean architecture.

## Components

### [Drawer](./Drawer)

Responsive drawer that works as a side panel on desktop and a swipeable
bottom sheet on mobile. Includes touch gesture handling with snap threshold,
body scroll lock, ESC key support, and CSS transitions.

**Tech:** React 18, TypeScript, CSS, ResizeObserver, Touch Events
